## Installation

1. Clone the repo

   ```sh
   git clone https://github.com/vn-aj-vngrd/unichem-ecommerce.git
   ```
2. Install NPM packages

   ```sh
   # Backend deps
   npm install
   ```
   ```sh
   # Frontend deps
   cd frontend
   npm install
   ```
3. Run Server

   ```js
   npm run server
   ```
4. Run App

   ```js
   npm run dev
   ```
